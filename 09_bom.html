BOM

window.open : 높이랑 너비를 지정해서 새 창을 띄움

window.setTimeout : 일정 시간 후 함수를 한번 실행 / id값 리턴

window.clearTimeout : setTimeout함수를 종료(setTimeout실행으로 리턴받은 아이디를 이용)

window.setInterval : 일정시간마다 함수를 반복실행 / id값 리턴

window.clearInterval : setInterval을 종료(id값 이용)

window.onload : 화면에 요소들이 로드가 완료되면 자동으로 한번 실행되는 함수

screen : client 운영체제 화면 속의 속성값을 갖는 객체

location : 브라우저의 주소표시줄과 관련된 객체

navigetor : 브라우저에 대한 정보
